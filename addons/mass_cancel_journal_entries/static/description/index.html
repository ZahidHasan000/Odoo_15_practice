Overview

  • This module helps to cancel or delete multiple journal entries from the journal entries tree view.

    • Features

    View multiple records of journal entries at once.

    Delete multiple records of journal entries at once.

    Once multiple entries are cancelled, all of their states changes to "Unposted".
